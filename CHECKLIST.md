# HdrTracer 전체 기능 점검 체크리스트

> 사용법: 새 버전을 설치한 뒤 이 목록을 따라 확인하고 `[ ]`를 `[x]`로 바꾼다.
> 평소에는 "이번에 고친 기능" 섹션만 확인하면 되고,
> 배포 직전에만 전체를 한 번 훑는다.

---

## 1. 설치 / 실행

- [ ] setup 실행 → 설치 마법사가 정상적으로 뜬다
- [ ] 설치 완료 후 "지금 실행" 체크 → 에러 없이 앱이 실행된다 (UAC 한 번은 정상)
- [ ] 시작 메뉴에 바로가기가 생겼다
- [ ] 바탕화면 바로가기가 생겼다 (설치 시 선택했다면)
- [ ] 바로가기 아이콘이 정상 표시된다 (흰 종이 아이콘 아님)
- [ ] 제어판 "프로그램 및 기능"에 `HdrTracer 1.0.0`으로 표시된다
- [ ] 바로가기 더블클릭 → UAC 한 번 → 앱 실행
- [ ] 앱이 켜진 상태(트레이 포함)에서 바로가기 다시 더블클릭 → 새 창이 안 뜨고 기존 창이 올라온다 (단일 인스턴스)

## 2. 인덱싱

- [ ] 첫 실행 시 "인덱싱 중" 표시가 나타났다가 사라진다
- [ ] 두 번째 실행부터는 캐시를 읽어 빠르게 시작된다
- [ ] 연결된 모든 NTFS 드라이브(C, D 등)가 검색 대상에 들어 있다 (하단 요약으로 확인)

## 3. 검색

- [ ] 파일 이름으로 검색이 된다 (Enter 또는 검색 버튼)
- [ ] 확장자 패턴 검색이 된다 (`*.jpg`, `*.png` 등)
- [ ] 여러 패턴 동시 검색이 된다 (`*.jpg *.png`)
- [ ] 검색 속도가 빠르다 (수십만 건도 1초 미만, 하단 ms 표시 확인)
- [ ] 결과 0건일 때도 앱이 정상이다
- [ ] 타이핑 중에는 검색이 실행되지 않는다 (Enter/버튼으로만)
- [ ] 검색 히스토리가 뜬다 (검색창 클릭/포커스 시)
- [ ] 히스토리 항목 클릭 → 그 검색어로 채워진다
- [ ] 히스토리 항목의 X(삭제) 버튼이 동작한다

## 4. 검색 결과 목록

- [ ] DRV / 이름 / 경로 / 크기 / 수정 날짜가 올바르게 표시된다
- [ ] 컬럼 헤더 클릭 → 그 컬럼으로 정렬된다 (다시 클릭 → 역순)
- [ ] 컬럼 경계선이 보인다 (DRV|이름, 이름|경로, 경로|크기, 크기|수정날짜)
- [ ] 경계선 드래그 → 컬럼 너비가 조절되고 모든 행이 같이 움직인다
- [ ] 크기/수정날짜 데이터가 왼쪽 정렬로 표시된다
- [ ] 휴지통($Recycle.Bin) 안의 항목은 결과에 안 나온다
- [ ] 숨김+시스템 폴더(예: !!QAdC) 안의 항목은 기본적으로 안 나온다

## 5. 결과 키보드 조작

- [ ] 검색창에서 ↓ → 결과 첫 항목으로 선택이 이동한다
- [ ] ↑/↓로 선택이 정확히 한 칸씩 이동한다 (길게 눌러도 맨 위로 점프하지 않는다)
- [ ] 첫 행에서 ↑ → 검색창으로 포커스가 돌아온다
- [ ] Enter → 선택한 파일이 열린다
- [ ] Delete → 휴지통 삭제 (확인 후)
- [ ] F2 → 이름 바꾸기
- [ ] Ctrl+Shift+N → 파일 이름 복사
- [ ] Shift+클릭 / Ctrl+클릭으로 다중 선택이 된다

## 6. 결과 우클릭 메뉴

- [ ] 열기
- [ ] 관리자 권한으로 실행
- [ ] 연결 프로그램으로 열기
- [ ] 폴더에서 보기 → 정상적인 탐색기 창이 뜨고, 해당 파일이 선택(강조)되어 있다
      (여러 번 반복해서 비정상 창/강조 실패가 없는지)
- [ ] 경로 복사
- [ ] 이름 복사
- [ ] 휴지통으로 삭제
- [ ] 이름 바꾸기
- [ ] 더블클릭 → 파일 열기

## 7. 실시간 반영 (USN)

- [ ] 파일을 새로 만들면 → 잠시 후 검색 결과에 나타난다
- [ ] 파일을 삭제하면 → 결과에서 사라진다
- [ ] 파일 이름을 바꾸면 → 새 이름으로 검색된다

## 8. USB (이동식 드라이브)

- [ ] USB를 꽂으면 자동으로 인덱싱되고, 현재 검색 결과에 그 USB 파일이 추가된다
- [ ] USB를 빼면 그 드라이브 항목이 결과에서 사라진다
- [ ] 설정에서 "이동식 드라이브 인덱싱"을 끄면 USB가 검색에서 빠진다

## 9. 설정 (Ctrl+, 또는 메뉴 → 설정)

- [ ] 설정 창이 열린다 (Ctrl+, / 메뉴 양쪽 다)
- [ ] 이동식 드라이브 인덱싱 토글 on/off가 동작한다
- [ ] "닫기 버튼을 누르면 트레이로 숨김" 토글이 동작한다
- [ ] "숨김·시스템 항목 표시" 토글: 켜면 !!QAdC 등이 결과에 나타나고, 끄면 사라진다
- [ ] 토글 변경 후 확인 → 검색 결과가 즉시 갱신된다
- [ ] 설정이 저장된다 (앱 재시작 후에도 유지)
- [ ] Tab으로 포커스 이동 시 포커스 사각형이 보인다 (회색 점선)
- [ ] Space로 토글 전환, Enter로 확인(저장 후 닫기)

## 10. 트레이

- [ ] X(닫기) → 앱이 트레이로 숨는다 (설정이 켜져 있을 때)
- [ ] 트레이 아이콘 더블클릭(또는 클릭) → 창이 다시 나타난다
- [ ] 트레이 우클릭 → 메뉴(열기/종료)가 뜬다
- [ ] 트레이 메뉴의 종료 → 앱이 완전히 종료된다 (트레이 아이콘도 사라짐)

## 11. 언어 (한국어 / 영어)

- [ ] 메뉴에서 언어를 영어로 전환 → UI 전반이 영어로 바뀐다
- [ ] 컬럼 헤더 (DRV/Name/Path/Size/Modified)
- [ ] 타이틀바 버튼 툴팁 (메뉴/최소화/최대화/닫기)
- [ ] 검색 버튼 툴팁 (검색 → Search)
- [ ] 히스토리 삭제 버튼 툴팁 (삭제 → Delete)
- [ ] 우클릭 메뉴 항목들
- [ ] 설정 창의 모든 텍스트 (토글 제목/설명, 확인 버튼)
- [ ] 다시 한국어로 전환 → 전부 한국어로 복귀

## 12. 창 동작

- [ ] 최소화 / 최대화 / 복원이 정상이다
- [ ] 창 크기 조절(가장자리 드래그)이 된다
- [ ] 타이틀바 드래그로 창 이동이 된다

## 13. 기타

- [ ] 메뉴(HdrTracer ▼) → 정보(About) 창이 뜬다
- [ ] 메뉴 → 단축키 창이 뜬다
- [ ] F5(인덱스 다시 빌드)가 동작한다
- [ ] 바탕화면에 진단 로그 파일(HdrTracer_*.txt)이 생기지 않는다
